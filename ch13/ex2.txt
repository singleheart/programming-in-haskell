a. left associative

                expr
       /         |       \
      expr       +      expr
  /    |     \           |
 expr  +   expr         term
  |         |            |
 term      term        factor
  |         |            |
factor    factor        nat
  |         |            |
 nat       nat           4
  |         |
  2         3

factor        term      term

  |            |         |

               |         |
               3         4
